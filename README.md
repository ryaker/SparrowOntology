# Sparrow Ontology

Ontology semantic layer for SparrowDB.

## What it is

Sparrow Ontology gives SparrowDB a semantic layer: type-safe entity creation, alias resolution, domain/range validation, and a pre-seeded world model that scales from a single person to a company of thousands.

## Status

Phase 1 (`sparrowdb-ontology-core`) in progress. See [implementation spec](https://linear.app/sparrowdb/project/sparrow-ontology-d0dd0956d1f0).

## Phases

- **Phase 1** (in progress): `sparrowdb-ontology-core` — the library
- **Phase 2**: `sparrowdb-ontology-mcp` — MCP server
- **Phase 3**: `sparrowdb-ontology-cli` — CLI binary
